Desarrollo móvil sin programación

Esta aplicación es un administrador de contactos la cual se encuentra en mi página web como una páagina de ejemplo Ver app, en está página te registras , después inicias sesión, y ahí podrás empezar a poner tus contactos con el botón de añadir. En la aplicación móvil que te encontrarás más adelante una vez que entres rota el teléfono.

Sistema operativo

Está aplicación es para el sistema operativo Android, no tiene extensión apk ya que el sitio que lo volvió app para que no te cobre lo hace de forma distinta.

Entorno de Desarrollo

El entorno de Desarrollo ideal para android es el IDE Android Studio, en este IDE se puede programar con Java o con Kotling, este es un IDE que también tiene emuladores cosa que muy importante en el desarrollo de aplicaciones porque tienes que asegurarte de que la app funciona en la mayoría de los dispositivos.
